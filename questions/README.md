## How to use this bank

1. **Cover the options and answer first.** The value is in *reasoning to* the ✅, not recognizing it.
2. **Read every "why" — including the wrong ones.** The distractors encode the exact traps interviewers set (bigger-model reflex, "it depends," reach-first RICE, thumbs-up-as-truth).
3. **For real-loop questions**, rehearse out loud in the shape the rubric rewards: *specific metric + specific failure mode + specific user.*
4. **Weight your prep by the real round weights** (Aakash Gupta, 47 placements): **Behavioral ~35% · Product Sense ~20% · Execution ~15% · Technical Depth ~15% · Presentation ~10%.** Most candidates over-prep product sense and skip behavioral — [start there](behavioral.md).
